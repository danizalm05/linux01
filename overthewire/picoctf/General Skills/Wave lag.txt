Problem Statement:

Python scripts are invoked kind of like programs in the Terminal... 
Can you run  this program:
     https://mercury.picoctf.net/static/325a52d249be0bd3811421eacd2c877a/ende.py
 using this passward:
    https://mercury.picoctf.net/static/325a52d249be0bd3811421eacd2c877a/pw.txt
 to get the flag
     https://mercury.picoctf.net/static/325a52d249be0bd3811421eacd2c877a/flag.txt.en
Hints 
  

Solution
  the script takes 2 arguments:
         a.   -e or -d to encrypt or decrypt the file, 
         b.   the name of file to encrypt or decrypt.
  So  we need run the script with the -d argument and the flag.txt.en file  
       python3 ende.py -d flag.txt.en
        output:
		  Enter password: 

   enter the password in the pw.txt file   we get the flg
